English

Opensource and distributed re-implementation old-shool browser game "Decline".
English language now no supported.

Русский

Ремейк браузерной игры начала нулевых "Decline".
Исходный код игры полностью открыт и игра не требует существования центрального сервера игры (точка-точка).
Для игры обязательным является наличие внешнего IP-адреса.

Установка

Скачать последнюю версию можно по ссылке https://github.com/nwtour/decline/archive/master.zip.
И распаковать в любой каталог

Windows

Для игры необходимо установить последнюю версию интерпретатора Perl
Желательно установить в каталог по умолчанию C:\Stawberry.
Запустить игру можно запустив game.bat в папке windows игры

Ubuntu Linux

Для установки необходимо выполнить команду в терминале:
sudo apt-get install libmojolicious-perl libcryptx-perl libdatetime-perl
Запустить игру можно командой в терминале:
perl КАТАЛОГ_ИГРЫ/game.pl

Первоначальная настройка

После успешной установки запустите игру и откройте в браузере http://127.0.0.1:3000
